통계적 우도(Likelihood, 가능도)는 관찰된 데이터를 바탕으로 특정 통계 모델의 모수(parameter)가 얼마나 그럴듯한지를 나타내는 척도입니다. 즉, 이미 발생한 데이터가 특정 확률 분포에서 나왔을 "가능성"을 의미합니다


BERT(Bidirectional Encoder Representations from Transformers)는 2018년에 구글이 공개한 사전 훈련된 모델입니다.
BERT는 2018년에 공개되어 등장과 동시에 수많은 NLP 태스크에서 최고 성능을 보여주면서 명실공히 NLP의 한 획을 그은 모델로 평가받고 있습니다.

InChI(International Chemical Identifier) 기반 화학 언어 모델은 분자의 구조 정보를 표준화된 텍스트 형식인 InChI를 활용하여 학습하고, 이를 통해 분자의 특성을 예측하거나 새로운 물질을 설계하는 전문 인공지능 모델입니다


1. SMILES (Simplified Molecular Input Line Entry System)
분자 구조를 텍스트 문자열로 표현하는 방식이에요. 간단하고 직관적이라 가장 널리 사용돼요.
예시: CCO (에탄올을 표현)
장점: 단순하고 이해하기 쉬움, 텍스트 기반이라 저장 공간 적게 차지
최신 트렌드: 2024년부터 AI 기반 SMILES 생성 모델이 신약 개발에 활발히 활용 중

2. InChI (International Chemical Identifier)
국제적으로 표준화된 화학 식별자로, 분자의 고유 "지문"과 같은 역할을 해요.
예시: InChI=1S/C2H6O/c1-2-3/h3H,2H2,1H3 (에탄올)
장점: 표준화되어 있어 동일 화합물의 다양한 표현을 통일할 수 있음
최신 트렌드: 2025년부터 양자 화학 계산 결과를 InChI에 통합하는 확장 버전 개발 중

3. MOL/SDF 파일
분자의 3D 구조와 원자 좌표를 포함하는 파일 형식이에요.
예시: (파일 형식이라 텍스트로 전체 표현은 어렵지만, 원자 좌표와 결합 정보를 포함)
장점: 3D 구조 정보를 완벽하게 보존, 추가 속성 데이터 포함 가능
최신 트렌드: 양자역학 기반 전자 밀도 정보를 통합한 확장 SDF 형식 등장 (2024년)

4. SELFIES (Self-Referencing Embedded Strings)
2019년에 개발된 비교적 새로운 표현 방식으로, 문법적으로 항상 유효한 분자 구조를 보장해요.
예시: [C][C][O][H] (메탄올의 SELFIES 표현)
장점: 기계학습에 최적화됨, 항상 유효한 분자 구조 생성
최신 트렌드: 2025년 생성형 AI 모델에서 SELFIES가 SMILES를 대체하는 추세
